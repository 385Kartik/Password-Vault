## Description
A simple command-line password manager in Python that allows users to store and retrieve their credentials locally using basic encoding for security.

## Features
- Add new credentials (site, username, password)
- View all stored credentials
- Base64 encoding for basic password security
- Data saved locally in a JSON file

## How to Run
1. Make sure Python is installed on your system.
2. Open terminal or command prompt.
3. Navigate to the project folder.
4. Run the program using:
```bash
python password_vault.py
```

Enjoy managing your passwords securely!
